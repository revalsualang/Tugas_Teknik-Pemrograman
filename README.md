## Deskripsi
Repository ini berisi tugas visualisasi data geofisika menggunakan Streamlit.

## File
- `seismic_app.py` → Visualisasi data seismik
- `magnetik_app.py` → Visualisasi anomali magnetik

## Cara Menjalankan
```bash
pip install streamlit numpy matplotlib pandas
streamlit run seismic_app.py
# atau
streamlit run magnetik_app.py
